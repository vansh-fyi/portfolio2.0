# Project Overview: Portfolio Website

## Project Name
Portfolio Website

## Purpose
This project serves as a personal portfolio website for Vansh, showcasing his skills, projects, and professional testimonials. It aims to present his work in a modern, interactive, and visually appealing manner.

## Executive Summary
The Portfolio Website is a static web application built with HTML, CSS (Tailwind CSS), and JavaScript. It features a responsive design, dark/light mode toggling, and various interactive UI components. The site is designed for direct static hosting, with external image assets. It includes sections for skills, projects, an "About Me" page, testimonials, and a contact form with a basic AI chat assistant.

## Technology Stack Summary
*   **Primary Languages**: HTML, CSS, JavaScript
*   **Styling Framework**: Tailwind CSS
*   **Iconography**: Lucide Icons
*   **Architecture**: Static Website (Monolith)

## Repository Structure
The project is structured as a **Monolith**, with all core application files residing in the `brownfield/` directory. Supporting assets are in `Images/`, and generated documentation will be in `docs/`.
